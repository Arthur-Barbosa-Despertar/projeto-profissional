# 🚀 TechNova Solutions

Projeto de site institucional desenvolvido para um trabalho de curso com o objetivo de praticar HTML e CSS na criação de páginas web modernas e responsivas.

---

## 📖 Sobre o Projeto

A TechNova Solutions é uma agência de tecnologia fictícia especializada em desenvolvimento de sites, sistemas e soluções digitais para empresas.

O site foi criado para apresentar:
- Informações da empresa
- Serviços oferecidos
- Planos e preços
- Formas de contato

---

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox
- Responsividade
